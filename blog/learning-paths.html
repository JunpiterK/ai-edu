Learning Paths

Read the archive like a field curriculum.

These paths turn a large article archive into a sequence: vocabulary first, then RAG operations, agent control, and runnable implementation.

Reading advice

Do not try to learn everything before building anything. Read one path, build a small slice, break it, then come back. The archive is designed so the second read makes more sense after the first mistake.
