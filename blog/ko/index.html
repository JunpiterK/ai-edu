한국어 저널

AI, 로봇, 온프레미스 LLM을 직접 만들고 운영하는 기술 블로그.

이곳은 클라우드 API 사용법만 모아둔 블로그가 아닙니다. 일반 게임용 GPU급 PC에서 로컬 LLM을 실행하고, RAG로 문서를 검색하고, MCP와 에이전트로 도구를 연결하며, 제조 현장과 로봇 AI에 적용하는 과정을 코드와 결과 화면까지 함께 기록합니다.

기본 언어는 영어입니다. 하지만 한국어도 영어판과 같은 범위의 정식 미러판으로 운영합니다. 스페인어, 중국어, 일본어, 독일어, 프랑스어는 준비 전까지 영어판으로 연결하고, 검수 기준이 확보되면 각 언어판으로 전환합니다.
