사례를 보는 법

모든 AI 아이디어가 성공한다고 주장하려는 사례가 아닙니다. 권한, 근거, 평가, 롤백처럼 데모와 신뢰 가능한 운영 시스템 사이에서 설계 압력이 생기는 지점을 보여줍니다.

로보틱스 AI 01

로봇 지능 스택: 인식, 정책, 엣지 추론

GPU·TPU·NPU와 모델 계열 강의를 로봇 시스템으로 확장합니다. 지연시간과 전력 제약 아래에서 로봇이 인식하고, 표현을 만들고, 계획하고, 행동하는 전체 흐름을 설계합니다.

온프레미스 LLM 02

제조 엔지니어를 위한 에어갭 지식 시스템

모델 가중치는 공장 안으로 들여오되 공정 데이터는 밖으로 내보내지 않는 엔드투엔드 아키텍처입니다. 플랫폼 선정, 오픈 웨이트 모델 전략, 데이터 계보, 사람 승인(HITL)을 함께 다룹니다.

아키텍처 청사진 모델 선택 매트릭스 보안 체크리스트
마스터클래스 기사 읽기
RAG · 에이전트 03

내부 문서를 위한 RAG·MCP 에이전트

청킹, 임베딩, 로컬 벡터 검색, 출처가 표시된 답변, MCP 도구 서버, 감사 추적, 사람 승인을 묶은 실무 스택입니다. 설명을 실행 가능한 시스템 설계로 전환합니다.

5개 파일 RAG 튜토리얼 MCP 서버 예제 HITL 워크플로우
RAG 데크 열기 에이전트 강의
팹 AI 04

반도체 AI: 가상 계측, 이상 탐지 및 제어
